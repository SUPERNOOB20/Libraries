Various C and Python libraries.

A [library](https://en.wikipedia.org/wiki/Library_(computing)) in programming is a file that contains tools to help you in your coding journey!
These libraries should be suitable for everyone as they already are, but you can always change their code to best suit your personal needs! :D


Here's a brief description of the tools each library provides you:

- **adaptive_screen_utils**: Extrapolates the HTML / CSS viewport concept (vw and vh) to Python. Comes in handy to adapt your GUIs to any screen resolution automatically :3
- **anim_utils**: Will provide utilities for animations (movement and opacity) (stretching tools aren't planned at the moment, but I will consider it in the far future)
- **collider_handler**: Provides triangular hitboxes.
- **colour_utils**: Provides methods to convert between RGB and HSV colour spaces. Will also provide a "colour spread generator".
- **geometry**: Provides some classes (objects) that model basic geometric tools. 
- **image_sequence_renamer**: Renames images in bulk (be careful, though! It will rename the WHOLE given directory!)
